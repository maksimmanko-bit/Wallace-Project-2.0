Shipping Portal Manual

Help & Instructions

Step-by-step guidance for every section of the portal.

Overview

Getting Started

The Shipping Portal is a shared record of freight received, shipped, picked up, inspected, or submitted for warranty. Changes saved by one employee are available to every user.

Check the connection first

Confirm that the header says Server Connected before creating or changing a record.

  1. Select the correct tab.Use the physical movement or task to decide where the record belongs.
  2. Search before creating.Search by PO, Job, BOL, customer, supplier, or another known reference to avoid duplicates.
  3. Open an existing record.Click the underlined PO Number, Job Number, or BOL # to read the complete record.
  4. Edit instead of recreating.Use the pencil button to change an existing record, then save the form.
Incoming Freight

Receiving

Use Receiving for goods ordered from suppliers and delivered to the warehouse.

Create an incoming shipment

  1. Click New Receiving.Enter PO Number, Supplier, Ordered By, ordered date and Classification.
  2. Add useful optional information.Description, ETA, carrier, tracking number, notes, and packing slip make the record easier to find and verify.
  3. Click Save Shipment.The new record appears in Receiving Logs with status Ordered unless another status was selected.

Receive the shipment

  1. Click Receive.Select the employee, confirm the date and time, and upload POD files or photos.
  2. Inspect before the driver leaves.If anything is damaged, short, missing, or incorrect, enable the OSD option.
  3. Document OSD.Describe the problem, take clear photos of freight, packaging and labels, and follow the carrier instructions shown on screen.
Never sign clean when an OSD issue is visible

Record the exception on the driver paperwork and keep packaging available for inspection.

ETA and status

ETA is optional and feeds Dashboard notifications. Incoming statuses are Ordered, Received, and Canceled.

Outbound Freight

Shipped Out

Use Shipped Out when freight leaves the warehouse for a customer, job site, or another destination.

  1. Click New Shipped Out.Customer, Carrier, Ship To Address, and Payment Type are required.
  2. Add Job Number and notes.Job Number becomes the clickable reference in the first table column.
  3. Add condition photos.Photograph the freight before it leaves so its outbound condition can be proven later.
  4. Edit from the pencil button.Saving an edit updates the same record and does not create a second shipment.

Payment Type options are Collect, Pre-Paid, and Wallace Covered. Shipped Out records can only be deleted by an administrator.

Customer Handoff

Customer Pick Up

Use this section when a customer collects freight directly from the warehouse.

  1. Click New Pick Up.Enter the required Job Number and select the employee who released the freight.
  2. Add customer and comments when useful.These fields are optional but help identify the handoff.
  3. Upload at least one photo.A handoff photo is required before the record can be saved.
  4. Open details from Job Number.The complete record and photos are available from the first column.

Customer Pick Up records can only be deleted from the Admin panel.

Trailer Movement

Trailers

Trailer Logs track the full movement from Expected through Arrived and Dispatched.

Required and optional information

Required: Trip #, BOL #, Stop #, and Status. Optional: Weight in lb and Trailer #.

  1. Expected.Create the record and attach searchable PDF or Excel supporting documents.
  2. Arrived.Change the status when the trailer arrives. The current date and time are recorded automatically, and the Email button prepares an arrival email.
  3. Dispatched.Change the status when the trailer leaves. The current date and time are recorded automatically.
  4. Review elapsed time.Trailer Logs show how long passed between Arrived and Dispatched.

Click BOL # to open the full record. Use Report an issue to add a description and photos.

Claims

Warranty

Use Warranty to create and manage warranty service claims. Everyone can create and edit claims; only administrators can delete them.

Base fields

Enter Job Number, Reported Issue, Order Type, Claim Type, Status, and any useful photos. Order Type is Install, Supply, or Stock. Claim Type is Damage/Defect, Shortage/Missing, or Other.

To be ClaimedDefault status. Record the issue and claim classification.
Working on itAdd Original Order ID, Customer Name, and Warranty Notes.
Request SubmittedKeep the original order, customer, and submission notes current.
ApprovedAdd Original Order ID, Customer Name, Warranty Notes, and Approved ID.
DeniedAdd Original Order ID, Customer Name, Warranty Notes, and Denied Reason.

Click Job Number for the full claim. The Dashboard summarizes warranty volume by period, status, and claim type.

Panel Quality

Panel Inspections

Use Panel Inspections to request a panel check and record the final inspection result.

Inspection RequestEnter Customer Name, Job Number, Notes, and select Requested By from the shared employee list.
Inspected - good to goSelect Inspected By and upload at least one required inspection photo.
Inspected - damage foundSelect Inspected By, describe findings in Notes, and upload required evidence photos.

The available inspectors are Bryan, Maksym, Alex, and Valerii. Everyone can create and edit inspections; deletion is admin-only.

Operations

Dashboard

Click Dashboard in the header to review activity for Today, Week, or Month.

  • Received, Shipped Out, and Picked Up totals.
  • OSD incident count and incidents by carrier.
  • Average time between order and receipt.
  • Overdue and upcoming ETA notifications.
  • Warranty claim count, claims by status, and claims by type.
Evidence & Retrieval

Search, Files & Reports

Search and filters

Search applies to the active tab. Use status and carrier filters where available. Trailer search also reads text stored inside text-based PDF and Excel attachments.

Photos and documents

Open a Photos, POD, OSD, Docs, or Issue button to launch the gallery. Use arrows to move between files, zoom or rotate images, and click Download to save the current file automatically.

PDF reports

Click the PDF icon in a record's Actions column. The downloaded report contains all saved information and labeled evidence photos such as POD, OSD, pickup, warranty, or inspection photos.

Photo uploads are optimized automatically

Large image files are reduced before upload to improve speed while keeping useful evidence quality.

Restricted Controls

Admin Panel

Click the gear icon and enter the administrator passcode. Admin changes are shared with all users.

  • Edit the main employee list and Customer Pick Up employee list.
  • Change receiving and trailer email recipients, subjects, and message templates.
  • Edit OSD instructions for each carrier.
  • Change the administrator passcode.
  • Search and permanently delete records, including Shipped Out, Customer Pick Up, Warranty, Panel Inspections, and Trailers.
Deletion is permanent

Confirm the reference carefully before deleting a record from the Admin panel.

Access & Troubleshooting

Mobile & Support

The portal works in current desktop, iOS, and Android browsers. Tables may scroll horizontally on small screens; forms and Save buttons remain inside the modal.

If something does not work

  1. Check Server Connected.Wait for the connection before saving.
  2. Refresh the data.Use the circular refresh button above the table.
  3. Check required fields.Fields marked with an asterisk must be completed.
  4. Keep the page open during uploads.Large groups of photos need time to finish uploading.
  5. Search for the record after saving.Do not press Save repeatedly while waiting for the first save to complete.
